# Damas-Chinas
Hecho por: Sofía Cristina Gómez Hernández

Juego de Damas Internacionales implementado en C++ con interfaz por consola.
Permite jugar entre dos jugadores (Blancas y Negras) usando coordenadas (ej.: B6 → C5). Incluye menú principal, reglas básicas de movimiento y captura, coronación de piezas y conteo de puntos.

Características principales:
Tablero de 10x10 (Damas Internacionales).
Piezas: B (Blancas), N (Negras); coronas: Q (reina blanca), K (rey negro).
Movimiento diagonal simple y capturas (saltos).
Coronación automática al llegar a la fila opuesta.
Conteo de capturas (puntos) y detección de victoria cuando un bando se queda sin piezas.
Menú inicial: 1. Jugar / 2. Salir.
Mensajes claros de error para coordenadas inválidas y movimientos no permitidos.
